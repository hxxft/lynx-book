#### 拓展 API

拓展 API 是 Lynx 在 JavaScript 运行环境下提供各个平台特殊能力的 JavaScript API。开发者可根据需求进行相关使用。



#### 拓展 API 概述

| 名称                                              | 功能                         |
| ------------------------------------------------- | ---------------------------- |
| [PageNavigator](../components/pagenavigator.md)   | 提供各个平台跳转页面的功能。 |
| [CoordinatorRegister](./interactive-animation.md) | 提供交互动画注册功能         |
| Storage                                           | 缺省                         |
| Camera                                            | 缺省                         |
| Calendar                                          | 缺省                         |
| Vibration                                         | 缺省                         |
| NetInfo                                           | 缺省                         |
